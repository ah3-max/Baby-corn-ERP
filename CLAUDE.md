# 玉米筍跨境供應鏈 ERP 系統 — 專案總覽

## 專案背景

本專案為 **玉米筍農產品跨境供應鏈公司** 開發的 ERP / CRM 系統。

公司主要業務：

泰國採購 → 工廠加工 → 出口 → 台灣倉儲 → 市場銷售 → 成本與利潤分析

產品為 **生鮮玉米筍（Baby Corn）**，具有：

- 保存期限短
- 批次品質差異大
- 成本結構複雜
- 出口物流流程長

因此本系統採用 **Batch（批次）為核心的供應鏈管理模式**。

---

# 系統定位

本系統不是傳統 ERP，也不是單純庫存系統。

系統本質為：

**農產品跨境供應鏈營運系統**

涵蓋四大營運區塊：

1️⃣ 泰國端採購與工廠管理  
2️⃣ 出口與跨境物流管理  
3️⃣ 台灣端倉儲與庫存管理  
4️⃣ 銷售、成本與利潤分析  

---

# 核心設計概念

整個系統必須以 **Batch（批次）為核心**。

每一批貨都必須能追蹤：

- 採購來源（農民 / 中盤商）
- 工廠收貨
- QC 檢查
- 包裝與加工
- 出口物流
- 台灣入庫
- 庫存狀態
- 銷售紀錄
- 成本結構
- 利潤分析

所有資料都必須能 **追溯到 Batch**。

# 開發策略

採用 **模組化逐步開發方式**。

開發原則：

- 每次只專注 **一個模組**
- 不一次生成整個系統
- 先完成 **Phase 1 MVP**
- 確認運作後再擴展 Phase 2

1️⃣ **Batch-Centric**
所有資料必須可追溯到批次
